# startsetup

##

Script to start setup.py and folders

### Install

- via pipx

```sh
pipx install git+https://github.com/phor3nsic/startsetup
```

- via pip

```sh
pip install git+https://github.com/phor3nsic/startsetup
```

```
|__ README.md
|__ setup.py
|__ .gitignore
|__ project_name/
    |__ __inity__.py
    |__ main.py
```
